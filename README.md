# Chess

A software to play chess developed in Java.

## How to run

This project is just a simple `Java` project, so compile it normally using a `JDK` (I recommend `OpenJDK 17`). The main class that contains the `main method` is `game.Main`

## Links

Click [here](https://julio-igreja.itch.io/chess) to download the game.
